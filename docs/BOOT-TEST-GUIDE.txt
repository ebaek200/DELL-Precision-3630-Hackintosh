============================================================
  1단계: USB 부팅 테스트
============================================================

1) 재부팅 → Dell 로고에서 F12 (부팅 메뉴)
2) "UEFI: USB" 또는 USB 디바이스 선택
3) OpenCore 부팅 피커 → macOS Ventura 선택
4) 부팅 완료 후 아래 체크리스트 확인


============================================================
  2단계: 부팅 후 체크리스트
============================================================

□ [기본] 이 Mac에 관하여 → "iMacPro1,1" 표시 확인
□ [오디오] 시스템 환경설정 → 사운드 → 출력 장치 확인, 소리 재생
□ [이더넷] 시스템 환경설정 → 네트워크 → 이더넷 연결됨 확인
□ [GPU] 이 Mac에 관하여 → 그래픽 → "Radeon RX 5700 XT 8GB" 확인
□ [USB] 전면/후면 USB 포트에 장치 꽂아서 인식 확인
□ [NVMe] 디스크 유틸리티 → 모든 디스크 인식 확인
□ [잠자기] Apple 메뉴 → 잠자기 → 5초 후 키보드/마우스로 깨우기
        → 정상 복귀되는지, 즉시 깨어남 없는지 확인
□ [CPU] 활성 상태 보기 → CPU 사용률 정상, 코어 6개 인식


============================================================
  3단계: 잠자기 심화 테스트
============================================================

터미널에서 실행:

  # 잠자기 최적화 설정
  sudo pmset -a sleep 10
  sudo pmset -a disksleep 10
  sudo pmset -a displaysleep 5
  sudo pmset -a womp 0
  sudo pmset -a proximitywake 0
  sudo pmset -a tcpkeepalive 0
  sudo pmset -a powernap 0
  sudo pmset -a hibernatemode 0
  sudo pmset -a standby 0
  sudo pmset -a autopoweroff 0

  # 설정 확인
  pmset -g

  # 잠자기 테스트
  sudo pmset sleepnow

  # 깨어난 후 로그 확인 (즉시깨어남 원인 추적)
  pmset -g log | grep -i "wake reason" | tail -5


============================================================
  4단계: 문제 없으면 → 내장 EFI 교체
============================================================

  ※ 이 작업은 반드시 USB 부팅 테스트 성공 후 진행

  1) 내장 EFI 파티션 마운트 (이미 /Volumes/EFI에 있을 수 있음)
     diskutil mount /dev/disk1s1

  2) 기존 EFI 백업
     cp -R /Volumes/EFI/EFI /Volumes/EFI/EFI_backup_MacPro71

  3) 기존 EFI 삭제
     rm -rf /Volumes/EFI/EFI/OC
     rm -rf /Volumes/EFI/EFI/BOOT

  4) USB EFI에서 내장으로 복사 (USB EFI = /Volumes/EFI\ 1/)
     cp -R "/Volumes/EFI 1/EFI/BOOT" /Volumes/EFI/EFI/
     cp -R "/Volumes/EFI 1/EFI/OC" /Volumes/EFI/EFI/

  5) 확인
     ls /Volumes/EFI/EFI/OC/config.plist

  6) 재부팅 (USB 제거 후 내장으로 부팅)

  7) iCloud 로그아웃 → 재로그인 (시리얼 변경됨)


============================================================
  5단계: iCloud 재설정
============================================================

  시리얼이 MacPro7,1 → iMacPro1,1로 변경되었으므로:
  1) 시스템 환경설정 → Apple ID → 로그아웃
  2) 재부팅
  3) 다시 로그인
  4) "이 Mac을 신뢰하시겠습니까?" → 신뢰


============================================================
  트러블슈팅
============================================================

▶ 부팅 안됨 (검은 화면)
  - F12로 내장 디스크 선택 → 기존 EFI로 부팅 (안전)
  - USB 부팅 실패는 내장 EFI에 영향 없음

▶ 오디오 안됨
  - alcid 변경 시도: boot-args에서 alcid=12 → 11, 13, 17, 27, 28
  - config.plist → NVRAM → boot-args 수정

▶ 잠자기 즉시 깨어남
  - 터미널: pmset -g log | grep "wake reason"
  - XHC (USB) → BIOS에서 USB Wake 비활성
  - GLAN (LAN) → BIOS에서 Wake on LAN 비활성
  - RTC → sudo pmset -a womp 0

▶ 이더넷 안됨
  - IntelMausi.kext가 로드되었는지 확인
  - 터미널: kextstat | grep -i mausi

▶ 커널 패닉
  - boot-args에 -v 추가하여 verbose 부팅
  - config.plist → NVRAM → boot-args:
    agdpmod=pikera alcid=12 keepsyms=1 debug=0x100 -v
